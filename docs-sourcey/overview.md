# ftpserverlib

`ftpserverlib` is a Go library for building custom FTP servers on top of an
[`afero`](https://github.com/spf13/afero) filesystem. It provides the FTP
protocol engine while the embedding application supplies authentication,
storage, TLS, and server policy through driver interfaces.

The library supports uploads and downloads, directory operations, active and
passive transfers, IPv6, TLS, resume, structured logging, and machine-readable
directory listings. Optional extensions cover hashing, allocation, available
space, symbolic links, transfer errors, and other server-specific behavior.

For a ready-to-run server, use the companion
[`fclairamb/ftpserver`](https://github.com/fclairamb/ftpserver) project. Use
this library when you need to embed FTP behavior into your own service.
